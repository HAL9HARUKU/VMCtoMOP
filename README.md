# VMC to MOP とは
VMCProcotol を受信して MOP(https://github.com/HAL9HARUKU/MopUE4) に反映するためのツールです。

VMCProcotol についてはこちらを参考にしてください。
https://protocol.vmc.info/

## 動作環境
- Windows 10 64bit

## 開発環境
- Unity 2020.3.4f1
